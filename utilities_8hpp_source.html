81 * Define QVector and QMatrix operator overloads.
82 * Note that QMatrix overloads don't simply use QVector
83 * overloads, since the complex vector dot product involves
84 * conjugation, which doesn't occur in complex matrix multiplication.

86 * (e.g. minus is plus(negative times)) since compiler optimisations
87 * may change the order of operations and confuse the overloads invoked.
88 * Definition of division using multiplication can furthermore
